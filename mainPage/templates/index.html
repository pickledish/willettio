what i'm working on

This Very Web Server

This website is hosted on a Raspberry Pi sitting in my basement running a Debian/Apache/Python stack. Just using Google Apps for mail right now, but I'm looking to move over to local Postfix system (if the Pi can even handle it!) soon.

Monte Carlo Matrix Analysis

Website created to show the output of a simulated-annealing time series analysis on random variables. When given samples of random variables, some will have higher than normal correlation with each other, and will thus be more effective together. This algorithm finds the best proportion of each to use.
